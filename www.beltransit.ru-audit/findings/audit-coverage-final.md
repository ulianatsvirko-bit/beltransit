# Итоговое покрытие аудита BelTransit

Дата итоговой сверки: 12 июля 2026 года.

Статусы:

- **Закрыто** — исправление внедрено и проверено.
- **Корректно без изменения** — замечание аудита не подтверждается в текущей версии или действие не требуется.
- **Осознанное решение** — задача рассмотрена, но внедрение противоречило бы фактической модели бизнеса.

## 1. Technical SEO

| Пункт аудита | Статус | Результат |
|---|---|---|
| Canonical, sitemap и robots использовали разные версии домена | Закрыто | Везде используется `https://www.beltransit.ru`. Поддомены и адрес без `www` перенаправляются на основной домен. |
| JSON-LD отсутствовал в исходном HTML | Закрыто | Разметка компании, услуг, FAQ, статей и хлебных крошек приходит в HTML до JavaScript. |
| Несуществующие страницы возвращали 200 | Закрыто | Несуществующие страницы и отсутствующие статические файлы возвращают настоящий 404 и запрет индексации. |
| Не хватало защитных HTTP-заголовков | Закрыто | Добавлены стандартные защитные заголовки. Строгий CSP и отдельная защита VPS отложены владельцем и не относятся к незакрытой SEO-ошибке. |
| Пустой HTML и единый JS-файл | Закрыто | 26 поисковых страниц создаются заранее; React, иконки и админка вынесены в отдельные файлы. |
| Нет длительного кеширования файлов | Закрыто | Хешированные ресурсы получают длительное immutable-кеширование. |
| Неправильные адреса OG/VK | Закрыто | Все социальные изображения используют основной `www`-домен. |
| Hreflang отсутствует | Корректно без изменения | У сайта нет отдельных языковых версий, поэтому hreflang не нужен. |

## 2. Performance

| Пункт аудита | Статус | Результат |
|---|---|---|
| Монолитный JS и блокировка первого отображения | Закрыто | Внедрены предварительный рендеринг и разделение файлов. Основной JS передаётся сжатым; админка не входит в основной пользовательский файл. |
| Нет immutable-кеша | Закрыто | Исправлено правилами Vercel. |
| Шрифты загружаются неоптимально | Закрыто | Убран CSS `@import`, добавлены ранние соединения с сервером шрифтов и `display=swap`. |
| Нет повторного замера | Закрыто | Мобильная production-сборка: FCP около 0,48 с, LCP около 0,48 с, load около 0,79 с, CLS 0, 10 запросов, около 304 КБ ресурсов. Google PageSpeed API был недоступен из-за исчерпанной суточной квоты самого API. |

## Итоговая проверка

- Production build и технический verify проходят.
- Предварительно создано 26 индексируемых страниц и 3 служебные страницы.
- Проверены 4 товарные страницы на ширинах 375, 414, 768, 1024 и 1440 px: 20 из 20 проверок прошли.
- У каждой страницы один H1, две разные CTA на товарных страницах, кнопки не ниже 44 px, горизонтального выхода за экран нет.
- Подтверждённый режим работы ежедневно с 10:00 до 22:00 добавлен в LocalBusiness schema.
